# Google Gemini AI Integration

## Get Your Free Google API Key

1. Go to: https://aistudio.google.com/apikey
2. Click "Create API key in new project"
3. Copy the API key
4. Add it to this file below as GOOGLE_API_KEY

## Setup

Replace the placeholder with your actual API key:

GOOGLE_API_KEY=your-api-key-here

## Features

- Automatic story generation using Google's Gemini AI
- Each scene has context-aware narration related to your prompt
- No subscription required (free tier available)
- Fast and reliable
